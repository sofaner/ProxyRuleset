# Clash规则/支持AI服务分流/适合开发者使用

**声明：** 本项目基于[ACL4SSR](https://github.com/ACL4SSR/ACL4SSR)大佬的规则修改，仅供学习使用！



## 重点分流介绍

| 分流名称   | 说明                                                         |
| ---------- | ------------------------------------------------------------ |
| 💬 国外_AI  | OpenAI、Claude、cursor、Gemini、Copilot 、Meta AI等合并在此，需美国节点的AI服务，一般这些节点都支持（新加坡/日本/韩国/美国/英国/德国） |
| 🐟 漏网之鱼 | 未命前面规则节点是走它                                       |

**同时支持以下分流：**

- 去广告
- 自动测速
- 微软分流
- 苹果分流
- 谷歌分流
- 媒体分流
- 游戏分流
